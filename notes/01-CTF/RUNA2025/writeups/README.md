# RUNA CTF 2025 Writeups

이 폴더에는 RUNA CTF 2025 문제 풀이를 Obsidian에서 바로 열 수 있도록 정리했습니다.

## 파일 설명
- `WRITEUP.md`: 대표 서술형 정리본
- `WRITEUP_SIMPLE.md`: 핵심 풀이만 빠르게 볼 수 있는 버전
- `RUNA2025_CTF_WRITEUPS.txt`: 카테고리/문제별 요약
- `RUNA2025_COMPLETE_WRITEUPS.txt`: 전체 풀이 모음 텍스트
- `RUNA2025_WRITEUP.tex`: LaTeX 버전

새로운 문제를 추가할 때는 [[../../../Templates/WRITEUP_TEMPLATE|템플릿]]을 복사하거나, 기존 파일을 참고해 `문제명.md`로 저장하세요.
